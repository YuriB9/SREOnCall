## REMOVED Requirements

### Requirement: Приём вебхуков Zabbix

**Reason**: Приём вебхуков Zabbix никогда не был реализован (в ingestion существуют только эндпоинты Alertmanager и Grafana); источник выводится из поддержки решением владельца продукта, чтобы спецификация не обещала несуществующую функциональность и система не позволяла создавать webhook-токены для источника без эндпоинта.

**Migration**: Существующие webhook-токены с `source: zabbix` рекомендуется отозвать через настройки тенанта. Командам, которым нужен приём из Zabbix, — направлять алерты через Prometheus Alertmanager / Grafana либо инициировать отдельное изменение с ADDED-требованием «Приём вебхуков Zabbix» и полноценной реализацией эндпоинта.
